# 📚 دليل MirrorBot-v3 المحسّن الشامل

## مقدمة

**MirrorBot-v3** هو بوت Facebook Chat متقدم وشامل يجمع بين نظام RPG معقد، نظام اقتصادي متطور، وتكامل كامل مع خدمات Google Gemini AI. تم تطويره ليصل إلى مستوى احترافي 100% مع أكثر من 20+ أمر متقدم وميزات أمنية شاملة.

---

## 🎯 الميزات الرئيسية

### 1. نظام الذكاء الاصطناعي (Gemini AI)

البوت يستخدم Google Gemini Pro و Gemini Pro Vision لتقديم خدمات ذكاء اصطناعي متقدمة:

| الأمر | الوصف | الاستخدام |
|------|-------|----------|
| `جيمني دردشة` | دردشة ذكية مع السياق | `جيمني دردشة ما هو الذكاء الاصطناعي؟` |
| `جيمني صورة` | تحليل الصور (Vision) | رد على صورة + `جيمني صورة ماذا ترى؟` |
| `جيمني رسم` | توليد صور من وصف | `جيمني رسم منظر طبيعي جميل` |
| `جيمني ترجم` | ترجمة النصوص | `جيمني ترجم Hello World` |
| `جيمني ملخص` | تلخيص النصوص | `جيمني ملخص [نص طويل]` |
| `جيمني قصة` | كتابة قصص قصيرة | `جيمني قصة مغامرة` |
| `جيمني كود` | كتابة أكواد برمجية | `جيمني كود دالة لحساب المضروب` |
| `جيمني شعر` | كتابة نصوص شعرية | `جيمني شعر الأمل` |

### 2. نظام الألعاب (RPG - Solo Leveling)

نظام RPG متقدم يشبه نظام Solo Leveling من الأنمي:

```
مغارة حالي          - عرض بروفايلك وإحصائياتك
مغارة بوابه [E-S]   - فتح بوابة غارة
مغارة انضمام        - الانضمام للغارة الحالية
مغارة هجوم [مهارة]  - قتال الوحش
مغارة متجر          - عرض الأسلحة والدروع
مغارة شراء [الاسم]  - شراء سلاح أو درع
مغارة تدريب         - التدريب اليومي
```

**الرتب المتاحة:**
- 🔵 **E-Rank**: بوابة سهلة (2000 HP)
- 🟢 **D-Rank**: بوابة متوسطة (8000 HP)
- 🟡 **C-Rank**: بوابة صعبة (25000 HP)
- 🟣 **B-Rank**: بوابة صعبة جداً (70000 HP)
- 🔴 **A-Rank**: بوابة قاتلة (250000 HP)
- ⚫ **S-Rank**: بوابة كابوس (1500000 HP)
- 🩸 **RED**: البوابة الحمراء (10000000 HP)

**الأسلحة المتاحة:**
- 🔪 خنجر صدئ (قوة 20)
- ⚔️ سيف الفارس (قوة 150)
- ⚡ نصل الرعد (قوة 600)
- 🗡️ خنجر باروكا (قوة 2000)
- 🐉 سيف كاميش (قوة 10000)
- 👑 نصل ملك الظلال (قوة 50000)

**المهارات القتالية:**
- 👊 **ضربة**: هجوم أساسي (1.5x الضرر)
- ✨ **تطهير**: استعادة 100 صحة
- 👥 **استدعاء**: جيش الظلال (5x الضرر)
- 🔥 **غضب**: مهارة نهائية (12x الضرر)

### 3. نظام الاقتصاد

نظام اقتصادي متكامل مع وظائف وتجارة واستثمارات:

```
اقتصاد حالي              - عرض حالتك المالية
اقتصاد عمل [نوع]        - العمل وكسب المال
اقتصاد بنك إيداع [مبلغ] - إيداع أموال
اقتصاد بنك سحب [مبلغ]   - سحب أموال
اقتصاد استثمر [نوع]     - استثمار أموالك
اقتصاد رهان [مبلغ]      - رهان بسيط
```

**أنواع الوظائف:**
| الوظيفة | الراتب | الخبرة | وقت الانتظار |

| عامل | 100$ | 10 XP | 1 ساعة |
| مهندس | 300$ | 30 XP | 1 ساعة |
| طبيب | 500$ | 50 XP | 1 ساعة |
| معلم | 250$ | 25 XP | 1 ساعة |
| صياد | 150$ | 15 XP | 1 ساعة |
| مزارع | 200$ | 20 XP | 1 ساعة |

**أنواع الاستثمارات:**

### 4. أوامر الترفيه

أوامر ترفيهية متنوعة تشمل ألعاب مصغرة وتحديات:

```
ترفيه كلمة      - لعبة تخمين الكلمة
ترفيه صح        - لعبة صح أم خطأ
ترفيه أرقام     - لعبة تخمين الرقم
ترفيه لون       - لعبة الألوان
ترفيه نكتة      - نكتة عشوائية
ترفيه حقيقة     - حقيقة مثيرة
ترفيه نصيحة     - نصيحة حكيمة
ترفيه تحدي      - تحدي يومي
ترفيه شخصية     - اختبار الشخصية
```

### 5. أوامر الإدارة

أوامر إدارية متقدمة للمطورين:

```
إدارة تشغيل           - تشغيل البوت
إدارة ايقاف           - إيقاف البوت
إدارة إحصائيات        - عرض إحصائيات البوت
إدارة حظر [معرف]      - حظر مستخدم
إدارة إلغاء_حظر [معرف] - إلغاء حظر مستخدم
```

### 6. نظام الأمان والحماية

نظام أمان متقدم يتضمن:

- **فحص الكلمات المسيئة**: الكشف عن الكلمات غير اللائقة
- **مكافحة السبام**: الكشف عن الرسائل المتكررة
- **نظام التحذيرات**: تحذير المستخدمين قبل الحظر
- **السجلات المتقدمة**: تسجيل جميع الأوامر والأخطاء
- **الحماية من الأوامر المريبة**: منع الأوامر الخطرة

## 🚀 البدء السريع

### 1. التثبيت والإعداد

```bash
# تثبيت المكتبات
npm install

# إعداد ملف الإعدادات
# تأكد من وجود config.json مع:
# - GEMINI_KEY: مفتاح Google Gemini API
```

```bash
npm start
```

عند استخدام البوت لأول مرة:

```

```

## 🐛 استكشاف الأخطاء

### المشكلة: البوت لا يستجيب

**الحل:**
1. تحقق من الاتصال بالإنترنت
2. تأكد من صحة `config.json`
3. تحقق من السجلات في مجلد `logs/`

### المشكلة: خطأ في Gemini API

**الحل:**
1. تحقق من صحة مفتاح API
2. تأكد من أن الحد المسموح به لم يتجاوز
3. جرب أمر بسيط مثل `جيمني دردشة مرحبا`

### المشكلة: بطء الأوامر

**الحل:**
1. تحقق من استخدام الموارد
2. قلل من عدد الأوامر المتزامنة
3. تحقق من السجلات للعثور على الأوامر البطيئة
