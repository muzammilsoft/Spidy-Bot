# 🕷️ Spidy Bot - مساعد فيالق انمي السودان 🇸🇩

بوت فيسبوك متطور يعمل بالذكاء الاصطناعي (AI Agent)، مصمم خصيصاً لمجتمع "فيالق انمي السودان" بالعامية السودانية.

## 🚀 المميزات الرئيسية
- **AI Agent (وضع الوكيل)**: البوت يفهم الرسائل الطبيعية ويقرر استخدام الأدوات المناسبة تلقائياً.
- **تعدد الأوضاع**:
  - `Hybrid` (هجين): يدمج بين الأوامر المباشرة والذكاء الاصطناعي.
  - `Normal` (عادي): يعمل بالأوامر التقليدية فقط.
  - `Universal` (عالمي): يستجيب لجميع الرسائل في المجموعات بشكل مختصر وعفوي (لضمان العمل في جميع المجموعات).
- **نظام صلاحيات متطور**:
  - مستخدم (0).
  - أدمن مجموعة (1).
  - مطور (2).
- **أوامر إدارية**: تجميد المجموعات، التحكم في البوت، شل (Shell) للمطورين.
- **ألعاب واقتصاد**: نظام RPG (مغارة الظلال)، زواج، ترفيه، واقتصاد متكامل.
- **توليد الصور**: تخيل أي صورة باستخدام ذكاء Pollinations.

## 📁 هيكلة الملفات
- `index.js`: نقطة الدخول الرئيسية والسيرفر.
- `handle/`: معالجات الرسائل والردود والتحكم.
- `script/commands/`: تضم كافة الأوامر المتاحة (اقتصاد، ألعاب، أدوات).
- `script/events/`: معالجة أحداث المجموعات (انضمام، مغادرة).
- `utils/`: الأدوات المساعدة (اللوغر، وكيل الذكاء الاصطناعي).
- `database/`: قاعدة بيانات SQLite لتخزين بيانات المستخدمين.

## 🛠️ التشغيل والتحميل (Deployment)

### 1. المتطلبات
- حساب فيسبوك (يفضل حساب تجريبي).
- ملف `appstate.json` (جلسة تسجيل الدخول).
- مفتاح API لـ Pollinations (موجود مسبقاً في الإعدادات).

### 2. الرفع على Render أو Vercel
1. قم برفع الكود إلى مستودع GitHub الخاص بك.
2. في **Render**: اختر "Web Service"، اربط المستودع، واستخدم الأمر `npm start`.
3. في **Vercel**: يدعم تشغيل Node.js، تأكد من إعداد Port 3000.
*البوت يحتوي على سيرفر Express مدمج لضمان عمله على هذه المنصات.*

### 3. الحفاظ على البوت نشطاً (Uptime Robot)
1. بعد تشغيل البوت على Render/Vercel، احصل على رابط الموقع الخاص بك (مثلاً: `https://my-bot.onrender.com`).
2. اذهب إلى [Uptime Robot](https://uptimerobot.com/).
3. أضف "New Monitor" من نوع "HTTP(s)".
4. ضع رابط البوت واجعل الفحص كل 5 دقائق.
*يحتوي البوت على آلية `Keep-Alive` داخلية تقوم بعمل ping لنفسها كل 5 دقائق.*
